## Run Locally

- git clone git@github.com:basir/node-react-ecommerce.git
- cd node-react-ecommerce
- Run Backend
  - npm install
  - npm start
- Run Frontend
  - open new terminal
  - cd frontend
  - npm install
  - npm start
